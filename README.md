  Physics Quest Pro — Интерактивный тренажер по физике


 Посмотреть проект онлайн:[http://physics.zavulv9r.beget.tech]

О проекте
Тренажер разработан для учеников старших классов. Он сочетает в себе теоретические материалы, базу задач с автоматической проверкой и игровые механики для повышения мотивации.

  Основные возможности:
- 32 темы: от Кинематики до Квантовой физики.
-  Умные задачи: генерация условий с разными числами.
-  Система подсказок: 4 уровня помощи (от наводящего вопроса до полного решения).
-  Мини-игры:"Гравитационный прыжок", "Электрическая цепь", "Законы столкновений".
-  Личный кабинет: статистика, уровень, история достижений.
-  Раздел ОГЭ/ЕГЭ: шпаргалки с основными формулами.



  Руководство пользователя

1. Начало работы
*   Регистрация: При первом входе нажмите кнопку «Создать аккаунт». Данные сохраняются в браузере.
*   Вход: Используйте Email и пароль, указанные при регистрации.

 2. Решение задач
1.  Перейдите в раздел «Темы».
2.  Выберите интересующий раздел (например, «Динамика»).
3.  Внимательно прочитайте условие.
4.  Если возникли трудности, нажмите кнопку « Подсказка».
5.  Введите ответ в поле и нажмите «Проверить».
6.  Если ответ верный — вы получите опыт (XP) и перейдете к следующей задаче.

 3. Игровой режим
В разделе «Игры» доступны симуляторы физических процессов:
*   Гравитация: рассчитайте скорость для мягкой посадки на Луну.
*   Электричество: подберите резистор, чтобы зажечь лампочку.
*   Столкновения: изучите закон сохранения импульса на примере бильярдных шаров.

 4. Профиль и статистика
На странице «Профиль» вы можете:
*   Загрузить аватар.
*   Посмотреть количество решенных задач и набранных очков.
*   Отправить отчет о прогрессе преподавателю (кнопка внизу страницы).



  Технологии
Проект разработан с использованием чистого стека веб-технологий:
*   HTML5 (структура)
*   CSS3 (стилизация, анимации, адаптивность)
*   JavaScript (логика, генерация задач, работа с LocalStorage)
