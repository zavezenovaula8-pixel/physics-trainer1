Подготовка к ОГЭ/ЕГЭ

Все формулы и теория для успешной сдачи!

ОГЭ Механика - основные формулы

Равномерное движение:

S = v · t
v = S / t

Равноускоренное движение:

v = v₀ + at
S = v₀t + at²/2
v² - v₀² = 2aS

Второй закон Ньютона:

F = ma

Закон всемирного тяготения:

F = G · m₁m₂/r²
ЕГЭ Электричество и магнетизм

Закон Ома:

I = U/R
P = UI = I²R = U²/R

Закон Кулона:

F = k · q₁q₂/r²

Сила Ампера:

F = BIl·sin(α)

Электромагнитная индукция:

ε = -ΔΦ/Δt
ОГЭ Термодинамика

Количество теплоты:

Q = cm(t₂ - t₁)

Уравнение состояния:

PV = nRT

Первый закон термодинамики:

ΔU = Q - A
ЕГЭ Оптика и квантовая физика

Формула линзы:

1/F = 1/d + 1/f

Энергия фотона:

E = hν = hc/λ

Фотоэффект:

hν = Aвых + Eк
ОГЭ Динамика: Законы Ньютона

Первый закон Ньютона (инерции):

ΣF = 0 → a = 0

Второй закон Ньютона:

F = ma

Третий закон Ньютона (действия и противодействия):

F₁ = -F₂

Сила тяжести:

Fₜ = mg

Сила упругости (закон Гука):

F = kΔx
ОГЭ Механическая работа и энергия

Механическая работа:

A = F·s·cosα

Кинетическая энергия:

Eₖ = mv²/2

Потенциальная энергия (гравитационная):

Eₚ = mgh

Потенциальная энергия (упругая):

Eₚ = kx²/2

Закон сохранения механической энергии:

Eₖ₁ + Eₚ₁ = Eₖ₂ + Eₚ₂
ЕГЭ Молекулярная физика

Количество вещества:

ν = m/μ

Уравнение состояния идеального газа:

pV = νRT

Основное уравнение молекулярно-кинетической теории:

p = (1/3)nm₀v²

Средняя кинетическая энергия молекулы:

Eₖ = (3/2)kT

Закон Дальтона:

p = p₁ + p₂ + ... + pₙ
ЕГЭ Термодинамика

Первый закон термодинамики:

ΔU = Q - A

КПД тепловой машины:

η = A/Q₁ = 1 - Q₂/Q₁

Работа газа при изобарном процессе:

A = pΔV

Количество теплоты при нагревании:

Q = cmΔT

Уравнение теплового баланса:

Q₁ + Q₂ + ... + Qₙ = 0
ОГЭ Электромагнитные явления

Закон Ома для полной цепи:

I = ε/(R + r)

Магнитный поток:

Φ = B·S·cosα

Закон электромагнитной индукции:

ε = -ΔΦ/Δt

Энергия магнитного поля:

W = LI²/2
ЕГЭ Колебания и волны

Период пружинного маятника:

T = 2π√(m/k)

Период математического маятника:

T = 2π√(l/g)

Уравнение гармонических колебаний:

x = A·sin(ωt + φ₀)

Скорость волны:

v = λ·f
ОГЭ Оптика

Закон преломления:

n₁·sinα₁ = n₂·sinα₂

Формула линзы:

1/F = 1/d + 1/f

Угол преломления:

sinγ = n·sinα
ЕГЭ Квантовая физика

Формула Эйнштейна для фотоэффекта:

hν = Aвых + Eк

Энергия фотона:

E = hν = hc/λ

Момент импульса электрона:

L = n·ħ

Энергия атома водорода:

Eₙ = -13.6/n² (эВ)
